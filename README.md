## The geometry

Nine interlocking triangles radiate from a single central point — the *bindu* — constrained so tightly that placing all vertices requires solving a nonlinear system with no closed-form solution. Ancient craftsmen achieved millimeter accuracy on bronze and stone using only compass and thread.

```
  ▲  * & = ~   →   4 Shiva triangles   (upward,   static)
  ▽  # @ % $ + →   5 Shakti triangles  (downward, dynamic)
  ○  O         →   bindu               (unity of opposites)
```

## The mathematics hidden inside

| Layer | Structure | Mathematical content |
|:-----:|-----------|---------------------|
| ①  | 9 triangles → 43 sub-regions | Overdetermined concurrency system |
| ②  | 16-petal outer lotus | Cyclic symmetry group **C₁₆** |
| ③  | 8-petal inner lotus | Cyclic symmetry group **C₈** |
| ④  | 3 nested squares (bhupura) | Projective boundary — finite ↔ infinite |
| ⑤  | Bindu | Fixed point of the full symmetry group |

> The triangle intersections are **not free parameters** — all nine must satisfy
> simultaneous concurrency conditions. Modern verification uses numerical root-finding.
> The *śilpa-śāstra* texts encoded the solution geometrically, millennia before
> the mathematics had a name.
